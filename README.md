## Installation 📦
1. Clone the repository: `git clone https://github.com/Waseemshahazad/DeviceSpec.git`
2. Navigate into the directory: `cd DeviceSpec`
3. Install dependencies: `npm install`

## Usage 🔧
To run the application, use: `npm start`
